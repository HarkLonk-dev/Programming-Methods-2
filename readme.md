Penalty Pro - Turtle Football Game

A simple penalty shootout game built with Python using the turtle graphics library. This project simulates a football penalty scenario with basic animation, scoring system, and two game modes.

Features
Player shooting animation
Goalkeeper with random dive behavior
Aim control using keyboard
Ball animation toward target
Two game modes:
1 Player (vs AI)
2 Players (turn-based)
Score tracking and round system
Simple user interface with instructions and game status
Controls
Press 1 to start 1 Player mode
Press 2 to start 2 Player mode
Use Left and Right arrow keys to aim
Press Space to shoot
Game Logic
The player aims and shoots the ball toward the goal
The goalkeeper dives with a random offset to simulate imperfect AI
A goal is scored if the ball is far enough from the keeper
The game ends after:
5 rounds in 1 Player mode
10 rounds in 2 Player mode
Technologies Used
Python 3
Turtle graphics
Random module
How to Run
Make sure Python 3 is installed
Run the game with:

python main.py
